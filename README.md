# NFT Game Formal Edition
## Backpack
### 功能
- 畫面下方會有10欄道具欄
- B鍵可以打開背包，背包開啟狀態下，ESC可關閉背包
- 使用滑鼠左鍵點擊道具或裝備
  - 當滑鼠沒有持有物品 && 被點擊的欄位有物品 -> 將物品拿起
    - 此條件下如按住shift則可將堆疊在一起的物品做分割
  - 當滑鼠持有物品 && 被點擊的欄位有物品
    - 兩者相同
      - 判斷該欄位還夠不夠放(這裡指該物品的堆疊上限)
        - 夠 -> 將滑鼠上的全部放進去
        - 不夠 -> 將該欄位補滿，剩餘的還在滑鼠上
        - 如被點擊的欄位物品已滿 -> 與手中的做交換
    - 兩者不同
      - 將兩者互換
